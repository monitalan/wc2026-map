Interactive map of all 16 FIFA World Cup 2026 stadiums across the USA, Mexico, and Canada.

Explore player challenges by venue — altitude, extreme heat, crowd noise, travel demands, and historic significance — alongside confirmed group stage matches and team assignments.

Built as a single-file HTML/JS/CSS app. No frameworks, no dependencies beyond Google Fonts.

## Features
- Clickable SVG map with all 16 stadium pins
- Filter by challenge type or host country
- Per-venue match schedule with confirmed teams
- Player intelligence notes for each stadium

## Data sources
FIFA, NBC Sports, Yahoo Sports, Sports Illustrated. Match data current as of May 2026.
